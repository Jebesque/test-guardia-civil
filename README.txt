App web móvil lista para usar con 423 preguntas extraídas de los PDFs.

Archivos:
- index.html -> app principal
- questions.js / questions.json -> base de preguntas
- app.js -> lógica
- manifest.webmanifest + sw.js -> soporte PWA/offline

Uso rápido:
1. Abre index.html en un navegador.
2. Para usarlo como "app" en el móvil, súbelo a GitHub Pages, Netlify o Vercel.
3. Luego abre la URL en el móvil y usa "Añadir a pantalla de inicio".

Incluye:
- temas 1 al 7
- modo examen
- modo aleatorio
- repaso de fallos
- guardado local de errores y marcados
